# urban_strike
URBAN STRIKE est un jeu de type Shoot 'em up développé entièrement en HTML5, CSS3 et JavaScript vanilla. Le joueur pilote un vaisseau et doit défendre une ville nocturne contre des vagues de drones ennemis envahisseurs. 
Touche : ← → Pour déplacer le vaisseau 
touche  : ESPACE  Pour tirer
Détruire tous les drones ennemis de chaque vague pour passer au niveau suivant. Survivez aux 3 niveaux pour remporter la victoire

Ce jeu a été développé volontairement pour la plateforme GamiX Hub — la plateforme officielle de jeux étudiants de l'école ESPRIT.

 GamiX Hub est une initiative d'ESPRIT School of Engineering permettant aux étudiants de partager leurs créations de jeux avec toute la communauté ESPRIT et au-delà.
