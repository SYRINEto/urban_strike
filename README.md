# urban_strike
URBAN STRIKE est un jeu de type Shoot 'em up développé entièrement en HTML5, CSS3 et JavaScript vanilla. Le joueur pilote un vaisseau et doit défendre une ville nocturne contre des vagues de drones ennemis envahisseurs. L'ambiance est cyberpunk urbaine — skyline nocturne, néons cyan et rouge, effets de scanlines CRT, explosions de particules.
